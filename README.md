## Deployment

This is a static site and does not require a backend.

### GitHub Pages

1. Push the repository to GitHub.
2. Open the repository settings.
3. Go to the Pages section.
4. Set the source to the main branch and the root folder.
5. Save the settings and wait for GitHub Pages to publish the site.

## Notes

- Open `index.html` directly or deploy it on GitHub Pages.
- All assets use relative paths inside `assets/`.
- No backend or build step is required."# lastweb"
